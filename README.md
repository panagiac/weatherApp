# WeatherApp
A Demo app by Giacomo Panattoni (panagiac@protonmail.com)
### What's Inside
- 100% Kotlin
- Clean Architecture (with an extra layer to the cake!)
- MVVM w/ LiveData + ViewModels
- Retrofit + RxJava2
- Dependency Injection w/ Koin
- Unit Test w/ Mock API
